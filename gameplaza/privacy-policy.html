1. Introduction

GamePlaza is an open-source game launcher application for Android. It allows you to:

  • Manage your game library
  • Track gaming activity and play time
  • Connect to PlazaNet social networks
  • Share gaming presence with friends

GamePlaza can connect to:

  • Official PlazaNet servers (accounts.plazanet.org) - Governed by this policy
  • Self-hosted PlazaNet instances - Different privacy policies apply

This Privacy Policy covers the GamePlaza app itself. Different privacy policies apply depending on which server you connect to.

2. Data the App Collects Locally

The following data is stored only on your device and is NOT sent anywhere unless you explicitly connect to a PlazaNet server:

2.1 Game Library Data

  • List of games you've added
  • Game titles and descriptions
  • Game cover art and banners
  • Play time statistics
  • Last played timestamps
  • Game installation status

2.2 App Preferences

  • Display language preference
  • Theme settings (dark/light mode)
  • Layout preferences

2.3 Optional Settings

  • PlazaNet server URL (if using self-hosted instance)
  • Crash report preferences
  • Background activity tracking settings

Your Control: All local data is stored on your device and remains under your control. You can clear or delete all data at any time through the app settings.

3. Data Sent to Servers (Optional)

The GamePlaza app ONLY sends data to a server if you explicitly:

  1. Set up a PlazaNet account
  2. Enter authentication credentials
  3. Activate presence tracking

Important: If you don't connect to a server, no data is sent anywhere.

3.1 When You Connect to a Server

The app sends the following to your chosen server:

  • Authentication: Username and authentication token
  • Gaming Presence: Currently playing game name (if applicable)
  • Activity Status: Online/playing/offline status
  • Device Type: Identifier that you're using GamePlaza (not personally identifying)
  • Timestamp: Last seen/activity timestamp

Important: Your authentication token and personal data are sent ONLY to the server you configured. The GamePlaza developers never receive this data.

4. When Connected to Official PlazaNet

When you connect GamePlaza to the official PlazaNet servers (accounts.plazanet.org):

4.1 What Happens to Your Data

Your data is governed by the PlazaNet Official Privacy Policy:

  • Data is stored securely with encryption
  • Data is not shared with third parties
  • Data is not sold to advertisers
  • You have GDPR rights (access, deletion, portability)
  • You can delete your account and data anytime

4.2 Data Retention

  • Account data retained indefinitely until deletion
  • Activity data retained during playtime
  • Server logs retained for 30 days

4.3 Your Rights with Official PlazaNet

You have the right to:

  • Access your data
  • Delete your account and data
  • Export your data in portable format
  • Correct inaccurate information

See the PlazaNet Privacy Policy for complete details and how to exercise these rights.

5. When Connected to Self-Hosted Servers

CRITICAL DISCLAIMER

This Privacy Policy does NOT apply to self-hosted PlazaNet instances. Different privacy practices apply.

5.1 What You Need to Know

PlazaNet's source code is open-source. Anyone can run their own PlazaNet server. If you connect GamePlaza to a self-hosted instance:

  • Different policies apply: Each server operator controls privacy
  • We don't operate it: The PlazaNet/GamePlaza team has no control over that server
  • We don't monitor it: We don't audit or verify self-hosted servers
  • We can't help: Contact the server operator for privacy issues, not us
  • You trust the operator: You're responsible for who you give your data to
  • We're not liable: Any data breaches or misuse by the operator are not our responsibility

5.2 Before Connecting to a Self-Hosted Server

You MUST:

  1. Verify HTTPS: Never connect to unencrypted (http://) servers, unless they're local
  2. Review Privacy Policy: Ask the operator for their privacy policy
  3. Verify HTTPS Certificate: Check the server has a valid SSL certificate
  4. Know the Operator: Only connect to servers run by people you trust
  5. Understand Risks: Acknowledge you're trusting that operator with your data

5.3 What Data the Server Operator Gets

Any self-hosted server operator will receive:

  • Your username and authentication credentials
  • Gaming activity and presence information
  • Your IP address (in server logs)
  • Basic server logs (timestamps, errors)
  • Everything else you share

5.4 Their Responsibility

The server operator is responsible for:

  • Securing your data
  • Protecting against breaches
  • Complying with data protection laws
  • Providing privacy policies
  • Responding to deletion requests

GamePlaza is not responsible for: Data breaches, privacy violations, data loss, or any misuse by self-hosted server operators. If you have concerns, contact the server operator, not us.

6. App Permissions

GamePlaza requests the following Android permissions. Here's why we need them:

6.1 Required Permissions

  • INTERNET: To connect to PlazaNet servers (only when you configure a server)
  • QUERY_ALL_PACKAGES: To detect installed games and apps on your device
  • READ_EXTERNAL_STORAGE: To access game artwork and media files
  • WRITE_EXTERNAL_STORAGE: To save game artwork and media files

6.2 Optional Permissions

  • POST_NOTIFICATIONS: To send you presence tracker notification (if enabled)

6.3 What We Don't Access

  • We do NOT access your contacts
  • We do NOT access your photos/gallery
  • We do NOT access your location
  • We do NOT access your call logs
  • We do NOT access your messages
  • We do NOT access your personal files
  • We do NOT access other apps' data

You Control Permissions: Since Android 6.0+, you can grant or deny permissions individually. Denying permissions may affect app functionality, but the app will still work.

7. Security

7.1 Local Device Security

  • Local Storage: Game library data is stored with standard Android file permissions
  • Encrypted Preferences: Sensitive settings use Android's EncryptedSharedPreferences
  • Token Storage: Authentication tokens stored securely

7.2 Network Security

  • HTTPS Only: All communication with servers uses HTTPS/TLS encryption
  • SSL/TLS 1.2+: Only secure protocols are supported

7.3 What You Can Do

  • Keep your device OS updated
  • Use a strong password for your PlazaNet account
  • Don't share your authentication credentials
  • Only connect to trusted servers
  • Verify server certificates are valid

No System is Perfect: While we implement strong security, no system is 100% secure. If you suspect a security issue, report it to security@plazanet.org immediately.

8. Children's Privacy

GamePlaza is not intended for children under 13 years old. If you are under 13, please get your parent's or guardian's permission before using this app.

We comply with COPPA (Children's Online Privacy Protection Act) and similar regulations. If a parent believes their child has provided information through GamePlaza, please contact us at privacy@plazanet.org.

9. Data Deletion

9.1 Local Data Deletion

You can delete all local data through:

  • App Settings: "Clear All Data" option
  • Android Settings: Settings > Apps > GamePlaza > Storage > Clear Storage
  • Uninstall: Uninstalling the app removes all local data

9.2 Account Data on Official PlazaNet

To delete your account from the official PlazaNet:

  1. Visit accounts.plazanet.org
  2. Go to Account Settings > Danger Zone > Delete Account
  3. Your data will be deleted within 30 days

9.3 Account Data on Self-Hosted Servers

The process will look the same as on official PlazaNet servers, but you must visit your specific self-hosted server.
